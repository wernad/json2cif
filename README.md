## Overview
This is utility tool created for ChannelsDB workflow related to CIF files (https://github.com/wernad/channelsdb). 
This application convert JSON containing PDB or Alphafill related data into CIF file. Output CIF file can be exported.
### Run the app

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) and either upload your JSON file or type in protein id and set corresponding database to convert JSON data to CIF data.

You can **compare** original JSON and generated CIF file.

You can also **download** generated CIF file.
